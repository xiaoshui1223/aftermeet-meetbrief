# Meeting Types｜会议类型

Classify with one primary type plus optional tags.

## Supported MVP Types｜MVP 支持类型

| 中文类型 | English Type | Signals｜判断信号 | Output Emphasis｜产出重点 |
| --- | --- | --- | --- |
| 播客 / 访谈 | Podcast / Interview | 多人对谈、主题驱动、议程较弱 | 话题地图、人物观点、关键句 |
| 嘉宾分享 / 主题分享 | Guest Sharing / Theme Talk | 嘉宾围绕主题讲经验、案例或方法论 | 观点拆解、案例、金句 |
| 训练营 / 课程回放 | Training Camp / Course Replay | 课程、作业、阶段学习、操作步骤 | 学习路径、步骤截图、练习指南 |
| 比赛动员 / 活动宣讲 | Competition Mobilization / Event Briefing | 奖励、时间线、提交方式、案例、号召行动 | 赛事信息前置、提交清单、行动指南 |
| 项目推进 / 待办同步 | Project Progress / Task Sync | 负责人、截止时间、阻塞、决策 | 决策、任务、负责人、截止时间、风险 |
| 工作汇报 / 阶段复盘 | Work Report / Stage Review | 结果、数据、问题、下一阶段计划 | 成果、问题、原因、下一步计划 |
| 需求讨论 / 方案评审 | Requirement Discussion / Solution Review | 方案选择、判断标准、通过或驳回 | 方案对比、决策依据、结论、后续跟进 |
| 产品演示 / 工具实操 | Product Demo / Tool Practice | 屏幕操作、命令、工具界面、重复步骤 | 步骤截图、操作路径、注意事项 |

## Mixed Meetings

Use tags when needed:

```text
嘉宾分享 Guest Sharing + 比赛动员 Competition Mobilization + 工具实操 Tool Practice
```

For mixed meetings, keep one main storyline and use tags to decide front modules and screenshot distribution.

## Type-Specific Front Modules｜类型专属前置模块

- 比赛动员 Competition: timeline, prizes, submission method, requirements, references.
- 项目推进 Project: decisions, tasks, assignees, deadlines, risks.
- 课程回放 Course: learning path, lesson map, assignments, practice checklist.
- 工具实操 Demo: tool goal, prerequisites, operation steps, common pitfalls.
