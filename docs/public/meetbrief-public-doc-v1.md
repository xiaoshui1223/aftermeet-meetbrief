# AfterMeet MeetBrief｜会后有数

高效制定会议纪要，沉淀复用会议资产。

把没人想重看的会议回放，整理成可检索、可复用的知识资产。

AfterMeet MeetBrief is an Agent-native Skill for Feishu/Lark meeting workflows. It classifies the meeting type, previews the writing strategy, captures key visual moments, and turns long meetings into structured briefs with mind maps, screenshots, knowledge cards, speaker insights, quotes, and action guides.

It is built for meetings where the screen matters: courses, live sessions, product demos, tool walkthroughs, interviews, project reviews, competition briefings, and team syncs.
