# Mini Camp Demo

This folder will contain the public demo for AfterMeet MeetBrief｜会后有数.

The first internal draft used Feishu AI Minutes embedded images only, which was not visually rich enough for a long meeting. The next demo should be rebuilt with browser capture.

Target for the detailed demo:

- Meeting type: guest sharing + competition mobilization + tool practice
- Duration: about 2h 21m
- Screenshot target: 35-60 candidate screenshots
- Final output: structured Feishu brief with mind map, key screenshots, speaker viewpoints, quotes, knowledge cards, and action guide
